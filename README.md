# miniDET

miniDET is a lightweight experimental implementation of a subset of the Deterministic Networking (DetNet) Packet Replication, Elimination, and Ordering Functions (PREOF).

## Features

- User-space implementation
- Packet Replication Function (PRF)
- Packet Elimination Function (PEF)
- containerlab topology
- Reproducible experiments
