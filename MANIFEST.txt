miniDET Artifact Manifest
=========================

Artifact Name
-------------
miniDET: A Lightweight Experimental Implementation of a DetNet PREOF Subset for Commodity IP Networks

Description
-----------
This artifact accompanies the paper:

    miniDET: A Lightweight Experimental Implementation
    of a DetNet PREOF Subset for Commodity IP Networks

It contains the source code, containerlab topology,
experiment scripts, Docker environment, and representative
experimental results used in the paper.

Contents
--------

Dockerfile
    Docker image definition for the experiment environment.

preof.clab.yml
    Containerlab topology definition.

sender-flow.py
    Packet sender.

receiver-flow.py
    Packet receiver.

prf.py
    Packet Replication Function (PRF).

prf-onepath.py
    Single-path PRF implementation.

pef-history.py
    Packet Elimination Function (PEF).

scripts/
    Experiment execution, analysis, and plotting scripts.

results/
    Representative experiment results, summary CSV files,
    generated figures, and environment information.

fig-minidet-topology.pdf
    Network topology used in the paper.

SHA256SUMS.txt
    SHA-256 checksums for integrity verification.

Requirements
------------
Ubuntu Linux
Docker
containerlab
Python 3.12
